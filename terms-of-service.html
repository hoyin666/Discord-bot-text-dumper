服務條款 (Terms of Service)

✅ 應用說明：

Text dumper 是一個快速批次發送訊息的 Discord 工具，旨在提升訊息發送效率。透過簡潔的斜線指令介面，使用者可便捷地設定訊息內容與重複次數（1-10 次），並透過確認按鈕執行發送。

1. 服務內容

Text dumper 提供以下核心功能：

  • 批次發送訊息：透過 /發送訊息 指令，使用者可設定訊息內容與重複次數（1-10 次），點擊確認按鈕後系統將按設定間隔（0.5 秒）發送訊息至當前頻道。
  • 私密確認介面：所有指令回應均為 ephemeral（僅使用者可見），保障操作隱私。
  • 可重複點擊按鈕：確認按鈕可重複點擊，無需重新輸入指令即可多次發送。
  • 黑名單管理：管理員可透過 /黑名單添加、/黑名單移除 與 /黑名單列表 指令管理禁止使用本應用的用戶清單。

2. 使用規範

2.1 禁止濫用行為

⚠️ 重要警告

請勿使用本工具用於 spam 炸群！

本應用設計目的為提升正常訊息發送效率，嚴格禁止以下行為：

  • Spam 攻擊：惡意大量發送重複、無意義或騷擾性訊息，干擾頻道正常運作。
  • 炸群行為：透過快速重複發送訊息淹沒頻道，影響其他用戶使用體驗。
  • 違反 Discord 服務條款：利用本工具從事任何違反 Discord 使用條款或社群準則的行為。
  • 騷擾或惡意攻擊：針對特定用戶或群組進行騷擾、威脅或仇恨言論傳播。

2.2 合理使用

使用者應確保本應用僅用於以下合法且合理的場景：

  • 在取得管理員或頻道擁有者同意的情況下，發送公告、通知或活動訊息。
  • 進行測試或示範功能時，於私人頻道或測試環境操作。
  • 遵守所在伺服器的規則與頻道主題，避免不當發送。

3. 黑名單政策

🛡️ 黑名單機制

維護管理者有權將違規用戶列入黑名單，禁止其使用本應用。

3.1 列入黑名單條件

以下行為將導致用戶被永久或臨時列入黑名單：

  • 經查證屬實的 spam 炸群行為。
  • 多次違反使用規範警告後仍持續濫用。
  • 利用本工具從事騷擾、詐騙或其他惡意活動。
  • 繞過黑名單限制（如使用多個帳號）再次濫用。

3.2 黑名單管理

  • 添加黑名單：管理員使用 /黑名單添加 user_id=[用戶ID] 指令將違規用戶加入黑名單，該用戶將立即無法使用本應用的任何功能。
  • 移除黑名單：若用戶承諾改正並提出申訴，管理員可使用 /黑名單移除 user_id=[用戶ID] 解除封禁。
  • 查看黑名單：管理員可透過 /黑名單列表 查看當前所有被封禁的用戶 ID。
  • 持久化儲存：黑名單資料儲存於 blacklist.json 檔案，即使 bot 重啟也不會遺失。

3.3 申訴流程

若用戶認為被誤加入黑名單或已改正行為，可透過以下管道提出申訴：

  • 電郵至 info@2o25shop.uk 聯絡管理員。
  • 提供用戶 ID、被封禁時間與詳細說明。
  • 管理員將於 3-7 個工作日內審核並回覆結果。

4. 免責聲明

Text dumper 僅作為工具提供服務，開發者與維護者不對以下情況負責：

  • 濫用後果：用戶因違反本條款或 Discord 服務條款而導致的帳號封禁、伺服器踢出或其他懲罰。
  • 訊息內容：用戶透過本應用發送的訊息內容由使用者自行負責，開發者不對任何違法、侵權或不當內容承擔法律責任。
  • 服務中斷：因 Discord API 變更、伺服器維護或不可抗力因素導致的服務中斷或功能異常。
  • 資料遺失：黑名單資料或其他設定因伺服器故障而遺失（建議定期備份）。

5. Discord 服務條款遵守

使用 Text dumper 時，使用者必須同時遵守 Discord 服務條款 與 社群準則。任何違反 Discord 政策的行為同樣構成對本條款的違反。
