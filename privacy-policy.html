隱私權政策

Text Dumper - Discord 批次訊息發送工具
最後更新日期：2025 年 10 月 7 日

1. 關於 Text Dumper

Text Dumper 是一個 Discord 應用程式，旨在提供快速批次發送訊息的功能。本應用透過 Discord 斜線指令（Slash Commands）運作，允許使用者在伺服器或私訊中重複發送指定訊息。

⚠️ 重要提醒：請勿使用本工具用於 spam 炸群行為。維護管理者有權將濫用者列入黑名單，禁止使用本應用。

2. 我們收集哪些資訊

為確保應用正常運作及防止濫用，Text Dumper 收集並處理以下最小必要資訊：

2.1 自動收集的資訊

  • Discord 用戶 ID：用於黑名單管理及權限驗證（例如：管理員功能）
  • 指令使用紀錄：記錄指令執行時間及類型（僅用於系統日誌，不包含訊息內容）
  • 伺服器 ID：用於識別應用安裝位置（僅當應用安裝於伺服器時）

2.2 不收集的資訊

  • 訊息內容：透過 Text Dumper 發送的訊息內容不會被儲存或記錄
  • 個人識別資訊（PII）：不收集姓名、電子郵件、IP 位址或其他個人資料
  • 瀏覽紀錄：不追蹤使用者在 Discord 外的活動

3. 資訊使用方式

收集的資訊僅用於以下目的：

  • 功能運作：執行批次訊息發送功能及互動按鈕
  • 濫用防護：維護黑名單以阻止炸群或濫用行為
  • 管理員功能：驗證管理員身份以執行黑名單管理指令
  • 系統維護：分析日誌以修復錯誤及改善效能

我們不會將您的資訊販售、租賃或分享給第三方。

4. 資料儲存與安全

Text Dumper 將黑名單資料（Discord 用戶 ID）儲存於 JSON 檔案中，並存放於安全的伺服器環境（DevBox）。我們採取以下措施保護資料：

  • 伺服器存取權限受密鑰保護（SSH Key Authentication）
  • 黑名單資料僅限授權管理員存取
  • 定期檢查系統日誌以偵測異常活動
  • 遵循 Discord 開發者政策及 API 使用條款

資料保留期限：黑名單資料將持續保留，直至管理員手動移除或應用停止運作。

5. Discord 資料使用政策

Text Dumper 完全遵循 Discord 開發者政策 及 Discord 服務條款。我們僅透過 Discord API 存取必要資訊，且不會：

  • 儲存或記錄私人訊息內容
  • 未經授權存取伺服器資料
  • 濫用 Discord API 或進行速率限制規避
  • 干擾 Discord 平台正常運作

6. 使用者權利

根據 GDPR 及資料保護法規，使用者享有以下權利：

  • 查詢權：查詢我們是否持有您的資料（聯絡我們提供您的 Discord 用戶 ID）
  • 更正權：要求更正不準確的資料
  • 刪除權（被遺忘權）：要求刪除您的黑名單紀錄
  • 限制處理權：要求暫停處理您的資料
  • 反對權：反對我們處理您的資料

若需行使上述權利，請透過以下聯絡方式與我們聯繫（需提供 Discord 用戶 ID 以驗證身份）。

7. Cookie 與追蹤技術

Text Dumper 不使用 Cookie、追蹤像素或任何網路追蹤技術。本應用為純 Discord Bot，僅透過 Discord API 運作，不涉及網頁瀏覽器或第三方追蹤。

8. 第三方服務

Text Dumper 託管於 ClawCloud（免費雲端環境），並使用以下第三方服務：

  • Discord API：用於機器人運作及指令處理（遵循 Discord 隱私權政策）
  • ClawCloud：雲端伺服器託管

我們不對第三方服務的隱私權政策負責，建議使用者自行查閱相關政策。

9. 兒童隱私

Text Dumper 不針對 13 歲以下兒童提供服務。我們不會故意收集未滿 13 歲兒童的個人資訊。若發現誤收集兒童資料，我們將立即刪除。

根據 Discord 服務條款，所有 Discord 使用者必須年滿 13 歲（部分地區為 16 歲）。若您未滿法定年齡，請勿使用 Text Dumper。

10. 政策變更

我們保留隨時修改本隱私權政策的權利。任何重大變更將在本頁面更新，並更新「最後更新日期」。建議定期查閱本政策以了解最新內容。

繼續使用 Text Dumper 即表示您同意變更後的政策。若不同意變更，請停止使用本應用並聯絡我們刪除您的資料。
