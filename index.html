📖 使用方法

  1. 1. 點擊加入

    點擊頁面任一「加入」按鈕，將 text dmuper 加到自己的 Discord。

  2. 2. 輸入斜線指令

    在任何地方輸入 /發送訊息。

  3. 3. 設定內容與次數

    填入要發送的內容與次數（1-10），送出後會顯示私密確認按鈕。

  4. 4. 點擊確認

    按下「確認發送」後，訊息將公開送到所在頻道；按鈕可再次點擊。
